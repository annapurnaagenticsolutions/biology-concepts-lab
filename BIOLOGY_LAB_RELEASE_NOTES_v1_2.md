# Biology Concepts Lab v1.2-stable — Release Notes

## Release intent

v1.2 moves the Biology Lab beyond a polished static prototype into a stronger teaching product. The main goal is to keep the same STEM Lab identity used by Math, Physics, and Chemistry while giving Biology its own learning behavior: observation, evidence, structure-function reasoning, systems thinking, and real-life transfer.

## Major additions

### 1. Concept Ladder

A new dynamic panel is generated for the selected lesson:

1. Name the concept.
2. Use visible or model-based evidence.
3. Explain the function or mechanism.
4. Connect the concept to a system.
5. Transfer the concept to a real-life case.

This prevents Biology from becoming a card-based memory page.

### 2. Teaching Depth + Board Readiness

A new age-aware explanation frame changes by class band:

- Class 1–2: observation and simple examples.
- Class 3–5: part-job-system reasoning.
- Class 6–8: mechanism and input-process-output thinking.
- Class 9–10: board-style concept, mechanism, diagram/application, and misconception correction.

### 3. Stronger mini-test feedback

The quiz result now gives:

- Whether the answer is correct.
- The correct option.
- The reasoning path.
- The next action for explanation quality.

### 4. Preserved v1.1 strengths

v1.2 retains:

- Class-level Biology Mission.
- Biology Field Notebook.
- Biology-specific visuals.
- Practice engine.
- Teacher, Guided, Student, and Assessment modes.
- Browser-local progress tracking.

## Deployment

Use `index.html` for standalone deployment or `biology_concepts_lab_stable.html` for the unified STEM Lab root.

## Constraints maintained

- Static HTML/CSS/JS only.
- No backend.
- No login.
- No external dependencies.
- No printable worksheet layer.
