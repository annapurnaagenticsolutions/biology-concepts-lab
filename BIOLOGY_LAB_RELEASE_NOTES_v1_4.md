# Biology Concepts Lab — v1.4-stable Release Notes

## Purpose

v1.4 is a visual-consistency and STEM-suite-alignment release. It keeps Biology's living-systems identity while making the lab feel more clearly connected to the existing Math, Physics, and Chemistry labs.

## Main enhancements

1. **Suite Design Alignment**
   - Added a visible shared-design layer for shell, controls, subject identity, and deployment handoff.
   - Aligned the desktop mode selector to the four-mode STEM Lab pattern.
   - Preserved the 2x2 mobile mode layout.

2. **STEM Connection Panel**
   - Each Biology lesson now includes Math, Physics, Chemistry, and Biology links.
   - This turns Biology into the fourth pillar of a connected STEM suite rather than a separate static page.

3. **Biology Visual Atlas**
   - Added a lightweight visual-system catalogue:
     - Living Evidence Sorter
     - Plant System Diagram
     - Animal Adaptation Board
     - Human Body System Map
     - Cell Comparison Viewer
     - Microbe World Panel
     - Food Chain Builder
     - DNA + Trait Visual
     - Conservation Balance Board
     - Field Notebook Lens

4. **Handoff polish**
   - Updated stable HTML copies.
   - Updated README.
   - Updated home-page Biology card patch.
   - Added explicit v1.4 stable file.

## Preserved from earlier versions

- Class 1–10 curriculum coverage.
- Eight-step lesson player.
- Student, Guided, Teacher, and Assessment modes.
- Field Notebook.
- Concept Ladder.
- Teaching Depth + Board Readiness.
- Guided Biology Investigation Studio.
- Living-System Simulator.
- Misconception Clinic.
- Browser-local progress.
- Static GitHub Pages deployment.

Use `index.html` for standalone deployment, or copy `biology_concepts_lab_stable.html` into the unified STEM Lab root.
