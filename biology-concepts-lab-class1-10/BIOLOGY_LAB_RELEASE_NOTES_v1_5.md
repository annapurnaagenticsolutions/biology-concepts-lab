# Biology Concepts Lab — v1.5-stable Release Notes

## Focus

v1.5 adds a Biology-specific Diagram Practice Studio while preserving the shared STEM Lab aesthetics and GitHub Pages static deployment model.

## Main additions

- Strand-aware diagram practice for Class 1–10 Biology.
- Label, match, sort, sequence, and scenario-style practice boards.
- Practice patterns for living evidence, plant systems, animal adaptations, human body systems, cells, microbes, food chains, environment, health, nutrition, heredity, evolution, and life processes.
- Interactive scoring with correct/review feedback.
- Answer-quality frame: structure → function → system effect.
- No printable worksheet layer; all practice stays inside the static web lab.

## Why this matters

Biology learning needs diagrams, classification, and systems thinking. v1.5 converts the Visual Atlas from a catalogue into an active practice surface so students can observe, label, explain, and transfer concepts.
