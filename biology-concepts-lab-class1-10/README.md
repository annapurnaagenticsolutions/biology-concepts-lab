# Biology Concepts Lab — Class 1–10 v1.9-stable

Static GitHub Pages-ready Biology learning lab aligned with the unified STEM Lab suite: Math, Physics, Chemistry, and Biology.

## What is included

- `index.html` — standalone entry point for this Biology Lab package.
- `biology_concepts_lab_stable.html` — stable filename for integration into the unified STEM Lab root.
- `biology_concepts_lab_v1_1_stable.html` through `biology_concepts_lab_v1_9_stable.html` — explicit version snapshots.

- `README.md` — deployment and integration notes.

Choose Class → Choose Chapter → Observe → Life View → Concept → System Rule → Why → Real Life → Try → Mini Test.

## Modes

- Student: clean self-paced lesson view.
- Guided: observation-first prompts.
- Teacher: teaching notes, delivery console, misconceptions, and classroom activity cues.
- Assessment: mini test, weak-strand dashboard, and class-band progress cues.

## Biology-specific visual identity

This lab keeps the same suite UI backbone as Math, Physics, and Chemistry, but uses Biology-specific visual systems:

- Living vs non-living evidence lens
- Plant parts and plant systems
- Animal habitats, adaptations, and classification
- Human body system maps
- Cell, tissue, and microbe views
- Food-chain, decomposer, biodiversity, and conservation views
- Heredity and evolution concept chains
- Field-notebook evidence framing
- Concept-ladder and board-readiness explanation scaffolding
- Biology Visual Atlas for visual-system consistency

## GitHub Pages deployment

### Standalone Biology deployment

Upload this folder to a GitHub repository and enable GitHub Pages. Open `index.html`.

### Unified STEM Lab integration

Copy `biology_concepts_lab_stable.html` into the root folder where Math, Physics, and Chemistry stable files are stored. Then add the Biology card from `stem_home_biology_card_patch.html` to the home page.

## Notes

- No backend.
- No login.
- No external dependencies.
- No printable worksheet layer.
- Progress is stored only in the browser using `localStorage`.
