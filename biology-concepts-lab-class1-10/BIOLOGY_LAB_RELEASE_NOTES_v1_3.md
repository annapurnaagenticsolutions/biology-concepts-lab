# Biology Concepts Lab v1.3-stable

## Purpose
v1.3 continues the Class 1–10 Biology Lab by strengthening the subject-specific teaching layer while preserving the shared STEM Lab design language used by Math, Physics, and Chemistry.

## Major additions

1. **Guided Biology Investigation Studio**
   - Converts each lesson into a small inquiry.
   - Adds investigation question, safe micro-lab, class-band method, and safety boundary.

2. **Living-System Simulator**
   - Adds an input → process → output model for each selected strand.
   - Covers Life Basics, Plants, Animals, Human Body, Health, Cells, Microorganisms, Ecosystem, and Genetics.

3. **Misconception Clinic**
   - Makes misconception correction explicit.
   - Gives evidence-based correction and better student-answer framing.

4. **Better age alignment**
   - Class 1–2: observe, name clue, sort, safe example.
   - Class 3–5: part/job/system reasoning.
   - Class 6–8: input-process-output reasoning.
   - Class 9–10: board-style mechanism + application + misconception correction.

## Package behavior
- Static HTML/CSS/JS only.
- GitHub Pages-ready.
- Browser-local progress through localStorage.
- No backend, login, paid layer, or printable worksheet positioning.
