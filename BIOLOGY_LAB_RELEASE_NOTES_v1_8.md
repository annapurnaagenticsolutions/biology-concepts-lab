# Biology Concepts Lab — v1.8-stable Release Notes

## Release focus

v1.8 is a teacher / presentation-mode polish release. It does not add heavy UI or new dependencies. It converts the strengthened v1.7 content layer into a more classroom-ready teaching flow.

## Added

- Teacher Delivery Console.
- Class-band-aware teaching sequence for Class 1–2, Class 3–5, Class 6–8, and Class 9–10.
- 2/5/8/4/3 minute lesson timing scaffold.
- Teacher board script for each selected Biology concept.
- Questioning ladder: Notice → Explain → Connect → Transfer.
- Exit-ticket options: one-line answer, diagram cue, and misconception correction.
- Stronger Teacher mode note inside the main lesson player.

## Preserved

- v1.7 Content Depth Studio.
- v1.6 Board Answer Builder.
- v1.5 Diagram Practice Studio.
- v1.4 STEM suite visual alignment.
- v1.3 Living-System Simulator and Misconception Clinic.
- v1.2 Concept Ladder.
- v1.1 Biology Field Notebook and class mission system.

## Deployment

Use `index.html` as the standalone page. Use `biology_concepts_lab_stable.html` for unified STEM suite integration. The explicit versioned copy is `biology_concepts_lab_v1_8_stable.html`.
