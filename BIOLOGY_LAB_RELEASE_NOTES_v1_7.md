# Biology Concepts Lab — v1.7-stable Release Notes

## Release focus

v1.7 is a Biology learning-quality release. It does not add a heavy new UI layer. It deepens the content, explanation quality, visual reasoning, real-life transfer, and misconception correction across the Class 1–10 Biology Lab.

## Major additions

### 1. Content Depth Studio

Every selected lesson now includes a new content-depth layer:

- expanded explanation
- purposeful visual reasoning prompts
- daily-life Biology questions
- class-wise explanation maturity
- better student answer guidance
- misconception check

### 2. Class-wise explanation maturity

The lab now adapts teaching depth by class band:

- Class 1–2: observe, sort, name a clue, and give a safe example
- Class 3–5: explain part → job → whole-system connection
- Class 6–8: explain input → process → output → evidence
- Class 9–10: use definition, mechanism, diagram cue, application, and misconception correction

### 3. Stronger biological reasoning

The new layer pushes students beyond definitions:

```text
What is it?
Where do we see it?
What structure is involved?
What function does it perform?
What system-level effect does it create?
What misconception should be avoided?
```

### 4. Purposeful visualisation

Visual prompts now clarify what students should draw, label, compare, sequence, or trace. The purpose is to support explanation, not decoration.

### 5. Real-life Biology questions

Each lesson now includes daily-life questions such as why plants bend toward light, why we breathe faster after running, why all microbes are not harmful, why decomposers matter, and why inherited traits create resemblance but not identical copies.
